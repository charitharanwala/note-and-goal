Todo Goals - GitHub PWA Version

This folder is only for the website/PWA version.
Upload the contents of this folder to GitHub Pages.

Files included:
- index.html
- github-todo-goals.html
- todo-goals.webmanifest
- todo-goals-sw.js
- todo-goals-icon.svg
- icon.png

How to use:
1. Upload all files in this folder to your GitHub repo.
2. Turn on GitHub Pages for the repo.
3. Open the GitHub Pages website URL.
4. On iPhone/iPad, tap Share, then Add to Home Screen.
5. On desktop Chrome/Edge, use the browser install option if shown.

Do not upload desktop files here.
This version does not need package.json, Electron, node_modules, or build scripts.
